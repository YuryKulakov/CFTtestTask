Общая информация
Версия Java - Java 11
Система сборки - Apache Maven 4.0.0

Используемые библиотеки:
Apache Commons Lang 3.12.0 (https://mvnrepository.com/artifact/org.apache.commons/commons-lang3/3.12.0)


Инструкция по работе
Пакет собран в .jar, сборка проекта - Maven.

Готовый .jar-файл находится: CFTtest\out\artifacts\CFTtest_jar.

Для запуска программы необходимо ввести следующие команды в cmd:

cd C:\Users\{NAME}\Desktop\CFTtest\out\artifacts\CFTtest_jar           (Указываем каталог ==> полный путь к файлу)

java -jar CFTtest.jar -s -a out.txt in1.txt                                  (Запускаем программу  с указанными опциями)

Где:

out.txt - имя выходного файла с результатом сортировки слиянием (Параметр обязательный), создается в той же папке, где лежит CFTtets.jar.

in[n].txt - имя входных файлов, количество которых должно быть не менее одного (Параметр обязательный), лежат так же в папке с CFTtets.jar файлом.

-a    Сортировка по возрастанию.
-d    Сортировка по убыванию.
-i    Сортировка целочисленных объектов.(Указывать обязательно)
-s    Сортировка строковых объектов.(Указывать обязательно)
